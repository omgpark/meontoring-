  • 멘토 출석 등록 → 활동일지 작성
  • 내 일지 조회(이름+생년월일 일치 시만 조회)
  • 관리자에서 미리보기/출력(PDF) 가능
사용 팁
• 날짜/생년월일은 YYYY · MM · DD 칸으로 입력하면 자동으로 다음 칸으로 넘어갑니다.
• 시간은 1530 입력 시 15:30으로 자동 변환됩니다.
※ 개인정보 보호를 위해 이름 + 생년월일이 일치하는 기록만 조회됩니다.
※ 기본 비밀번호: 1234

출력 옵션

• 출력은 “요약 1페이지 + 선택된 일지들(페이지 자동 분리)”로 생성됩니다.
